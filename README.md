## üöÄ Getting Started

### Prerequisites

Before starting these exercises, ensure you have the following installed:

- **Git** - [Download Git](https://git-scm.com/downloads)
- **GitHub Account** - [Sign up for free](https://github.com/signup)
- **Text Editor** - [VS Code](https://code.visualstudio.com/) (recommended), Notepad++, or any code editor
- **Web Browser** - Chrome, Firefox, Edge, or Safari
- **.NET SDK** (for C# exercises) - [Download .NET](https://dotnet.microsoft.com/download)
- **Node.js** (optional, for some exercises) - [Download Node.js](https://nodejs.org/)

### How to Use This Repository

1. **Read the exercises** - Each exercise is a markdown document with detailed instructions
2. **Create separate repositories** - Each exercise should be done in its own GitHub repository
3. **Follow the workflow** - Practice proper Git workflow with branches and PRs
4. **Complete exercises in order** - They build upon each other progressively

### Typical Workflow for Each Exercise

```bash
# 1. Create a new repository on GitHub for the exercise
# 2. Clone it to your local machine
git clone https://github.com/YOUR-USERNAME/exercise-name.git
cd exercise-name

# 3. Create a branch for your work
git checkout -b feature/implement-exercise

# 4. Complete the exercise following the instructions

# 5. Commit your changes
git add .
git commit -m "Implement [feature description]"

# 6. Push to GitHub
git push -u origin feature/implement-exercise

# 7. Create a Pull Request on GitHub

# 8. Review and merge the PR

# 9. Move to the next exercise
```
